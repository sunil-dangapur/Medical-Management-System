
# Medical Management System

## 📌 Project Description

A full-stack web application developed using Django to manage medical store operations such as inventory management, customer handling, and billing system.

## 🚀 Features

* Add, update, and delete medicines (CRUD operations)
* Customer and order management
* Role-based access (Admin & User)
* Purchase and billing management
* Dashboard for tracking activities

## 🛠 Tech Stack

* Backend: Django (Python)
* Database: SQLite
* Frontend: HTML, CSS, Bootstrap

## ▶️ How to Run

1. Clone the repository
2. Install dependencies

   ```
   pip install -r requirements.txt
   ```
3. Run the server

   ```
   python manage.py runserver
   ```
4. Open in browser

   ```
   http://127.0.0.1:8000/
   ```
